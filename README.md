# random-password-generator
Will generate specified number of passwords:

```shell
NAME:
   genpwd - generate random passwords of specified length

USAGE:
   genpwd [global options] command [command options] [arguments...]

VERSION:
   0.0.0

COMMANDS:
     help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --length value, -l value   Password length (default: 16)
   --number value, -n value   Number of passwords to generate (default: 1)
   --digits value, -d value   Number of digits in password (default: 4)
   --symbols value, -s value  Number of symbols in password (default: 2)
   --upper, -u                Use upper and lower case
   --repeat, -r               Repeat characters
   --help, -h                 show help
   --version, -v              print the version
```
